# contact-qrcodes

QR contact card assets for the LSJ Global team.

## Structure

- `vcards/`: published `.vcf` contact files
- `qr/`: generated QR SVG files
- `src/main/java/com/lsjglobal/`: QR generator sources

## Notes

- Active staff contact cards are kept in the repository.
- QR links target GitHub Pages under `/vcards/...`.
